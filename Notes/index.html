HTML Basics

The HTML begins with typing "html" in atom and clicking enter. It'll come up with the language, the direction, the charset (the encoding) etc.. The title is what you want the title of the webpage to be displayed in the top tab of the website. This is inside the "head" part of the HTML. Besides this the html consists of the head and the body.

Also, to use italics and bold, use the "em" and "strong" tag respectively. Both are closed tags - meaning they have an opening <> and a closing <>. Things like the Horizontal rule (hr) and (br) are open tags so only need one set of <>.

Next, there are other things that can be implemented / added such as images, links and tables. To insert a image, type "img" and click enter. The source will be the source of the image. Eg: if the img is added to the folder, then look at the hierarchy. Eg: images/image.png

To add a link, use the anchor key - "a". The "href" will be the link and the text you put in between the anchor brackets is the displayed text. Eg: This is a link to the colour pallettes

Writing a table involves writing "table". Each row, write a "tr" and inside, each "td" is a cell. The first row can be the "th", table header. Example:

The next thing to learn is forms. Forms is created by typing forms and pressing enter on atom. It gives an "action" and a "method". Action refers to where the form leads you when you press the submit button. Method refers to what you want the form to do when you press the submit button.

The form needs a "label" to label the title for that entry. The form also needs an "input" to input things into the form. There are different input types. Eg: "input type="text" / "email" / "submit" etc... The input type submit creates a button. You can even enter "textarea" to create a text field.
